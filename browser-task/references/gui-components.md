# GUI 交互组件识别与操作

本文件覆盖常见网页交互组件的识别、边界判断和操作验证。

## 通用原则

1. 边界清晰化：明确组件的视觉边界和操作区域
2. 可见性优先：确保目标组件在可见区域内再操作
3. 逐步验证：每次操作后观察 UI 反馈确认成功
4. 自适应缩小：连续失败时减小操作幅度

## 失败处理流程

操作失败 → 重试 1 次 → 重新定位 → 重试 1 次 → 缩小范围慢速查找 → 重试 1 次 → 仍失败 → 请求人工接管

## 组件类型与操作

### 按钮类

**识别特征**：矩形框内包含文字或图标，鼠标悬停有高亮效果

**操作方法**：
- 点击中心位置
- 观察反馈（颜色变化、弹窗出现、页面跳转）

**常见问题**：
- 按钮被遮挡：滚动避让或关闭遮挡物
- 点击无反应：等待加载完成后再试

### 输入框类

**识别特征**：矩形框内有光标闪烁，可能有占位符提示文字

**操作方法**：
- 先聚焦（点击输入框）
- 再输入内容
- 按 Enter 或点击提交按钮

**常见问题**：
- 无法聚焦：检查是否有弹窗遮挡
- 输入失效：重新聚焦后再输入

### 下拉菜单

**识别特征**：显示当前选项，右侧有向下箭头

**操作方法**：
- 点击展开菜单
- 选择目标选项
- 验证选中状态（菜单收起后显示新选项）

**常见问题**：
- 菜单打不开：尝试多次点击或检查是否被遮挡
- 选项不生效：重新展开并再次选择

### 滑块

**识别特征**：水平或垂直轨道上有可拖动的手柄

**操作方法**：
- 按住手柄
- 缓慢拖动到目标位置
- 松开鼠标

**常见问题**：
- 拖动不灵敏：减慢拖动速度
- 超出范围：回退到有效范围内

### 复选框/单选框

**识别特征**：小方框或小圆圈，复选框可多选，单选框只能单选

**操作方法**：
- 点击框体本身（非文字）
- 验证选中状态（出现对勾或填充）

**常见问题**：
- 点到文字没反应：改点框体
- 取消选中：再次点击即可

### 标签页

**识别特征**：一排横向排列的标签，当前激活的标签有特殊样式

**操作方法**：
- 点击目标标签
- 验证内容更新（页面内容切换）

**常见问题**：
- 切换无效：等待加载完成
- 内容不显示：刷新页面

### 分页

**识别特征**：显示"上一页/下一页"或页码数字

**操作方法**：
- 点击目标页码或"下一页"
- 验证内容更新（列表项变化）

**常见问题**：
- 翻页无变化：检查是否已到最后一页
- 加载超时：等待或刷新重试
