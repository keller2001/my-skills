# 时效性信息获取

## 核心原则
1. **严禁主观断定或推测结论**：所有信息必须来自当前有效数据源，禁止基于已有知识或经验进行猜测。
2. **优先获取最新官方信息**：
   - 需要最新数据（价格、教程、动态等）时，主动按时间筛选，若网页内无按时间筛选，还可以添加时间限定词（如"最新"）
   - 优先选择官方网站、权威媒体或平台认证账号的信息
3. **验证链接有效性**：确保提供的链接可访问，避免失效或错误链接。

## 执行步骤
1. **明确时间范围**：
   - 若用户未指定时间，默认获取最近3个月内的信息
   - 对教程类内容，优先选择版本号最新的资源
2. **筛选数据源**：
   - 使用时间排序功能（如"按时间从新到旧"）
   - 排除明显过时的内容（如2年前的教程）
3. **验证信息有效性**：
   - 检查链接是否能正常打开
   - 确认信息来源的权威性
   - 交叉验证关键数据（如价格、日期）
4. **标注时效性**：在输出中明确标注信息的获取时间和来源。

## 特殊任务
1. **用户要求返回指定软件的安装教程**
- 若官方提供了安装教程，需要寻找到含有该教材的网页，将教程和网页链接一并返回
- 若官方未提供安装教程，需要下载指定软件，亲自安装一遍，直至第一次安装成功，返回安装步骤

## 常见问题处理
- **信息过时**：若发现信息已过期，立即寻找替代来源3
- **链接失效**：尝试查找该资源的最新版本或替代链接
- **数据冲突**：优先选择官方发布的数据
