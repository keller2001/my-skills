# 创作发布工作流（仅“文字配图”发帖：文字做成图片）

目标：在“收集整理”的素材基础上或直接按用户提供内容，完成有原创表达的笔记创作，并在获得用户明确授权且当前环境可操作时，用“文字配图”完成发帖发布。

## 适用范围（不满足则直接中止并告知原因）

- 仅当用户需求表达为：想发一个“文字是图片/大字图/文字配图”的图文笔记时，才执行本工作流
- 若用户提供图片/视频，或要求我搜索下载图片并上传发布：直接停止并回复当前不支持上传图片/视频；可改为“文字配图”或由用户手动上传后我再继续填标题/正文/标签

## 输入信息（缺什么先补什么）

- 主题与目标：想解决谁的什么问题（选购/避坑/路线/清单/体验）
- 受众画像：新手/进阶、预算区间、肤质/场景、城市/出行天数
- 素材来源：来自收集文档、你的浏览记录、你提供的图片/视频、或你提供的链接/截图
- 风格偏好：干货清单 / 体验叙事 / 对比测评 / 省钱攻略
- 发布约束：不能写的点（合规/品牌/敏感词），是否需要“更克制/更夸张”

## 工作流

### 1) 选角度与差异化

- 从素材里挑 3-5 条“高价值点”（方法、坑点、结论、清单、对比维度）
- 定义你的差异化：
  - 更具体：用数字、步骤、条件、场景限定替代空泛形容
  - 更可执行：把经验转成“怎么做/怎么选/怎么安排”
  - 更可信：明确前提与边界（人群/预算/季节/肤质/机型）

### 2) 产出大纲（先结构后文案）

推荐结构（按需选用）：

- 开头：一句话结论 + 适用人群/前提
- 正文：3-6 个小节（对比维度/路线安排/步骤清单/避坑要点）
- 补充：预算/替代方案/注意事项/常见问答
- 收尾：总结 + 行动建议 + 互动引导（评论区提问）

### 3) 改写规则（避免“拼贴感”）

- 不直接复用原笔记的句子与段落，使用自己的表达重写
- 保留事实与逻辑，但重组叙事顺序与表达方式
- 给出你的判断依据与使用前提，避免绝对化、医疗化与夸大承诺
- 引用对标内容时，以“思路/结构/要点”复用为主，不复用原文表达

### 4) 生成可发布内容包（标题/正文/标签严格分离）

至少包含：

- 标题候选 5-10 个（覆盖：结论型/避坑型/清单型/对比型/人群型）
- 正文 1 版主稿 + 1 版更短更利落的备选
- 封面文案建议（3-5 条，控制在短句）
- 标签矩阵（10-20 个：大词 + 场景词 + 人群词 + 城市/机型/肤质等限定词）
- 互动引导（2-3 条）：提问、投票、求建议、想看下一篇

### 5) 文字配图（大字图）生成

约束：当前仅支持在“文字配图”模块生成一张大字图用于发布，不支持上传用户本地图片/视频。

- 生成 1 张大字图方案：
  - 大字文案（10-18 字，强结论/强利益点）
  - 小字补充（20-40 字，限定人群/场景/边界）
  - 版式建议（居中/左对齐）、配色建议（高对比易读）
  - 背景风格（纯色/渐变/轻纹理），避免信息噪音

### 6) 发布（在获得用户明确授权后）

原则：默认只提供“怎么做”的步骤与准备清单；只有在用户明确授权后，才执行账号内动作，并回传每一步结果。

#### 6.1 授权与前置检查

- 明确要执行的动作：发帖（文字配图）
- 若用户未登录或当前不可操作：改为输出“手动操作步骤 + 检查点”，不要臆造已执行结果

#### 6.2 发帖（文字配图）

- 发布前检查：
  - 标题、正文、标签、地点（如适用）齐全
  - 合规限制已满足（避免敏感词/极限词/医疗化表述等）
  - 已在“文字配图”模块生成大字图（无需上传本地图片/视频）
- 输入顺序与字段分离（防止 # 混入正文导致输入错误）：
  - 先把标题输入到标题栏（只放标题，不含任何 # 标签）
  - 再把正文输入到正文栏（正文里不要写 # 话题标签）
  - 最后在标签/话题入口单独添加 3-8 个标签（每个标签单独添加；不要把 #tag 作为正文的一部分粘贴）
- 发布步骤（按平台 UI 顺序）：
  - 进入发帖 → 选择“文字配图” → 生成大字图 → 填标题 → 填正文 → 单独添加标签/话题与地点 → 预览检查 → 发布
- 发布后回传：
  - 发布状态（成功/失败/需手动）与原因
  - 发布链接（如可获得）
  - 建议的首小时互动策略（例如：置顶评论/引导提问）

## 场景模板

### 评测类（3C/美妆）发布模板

- 标题：结论 + 对比对象 + 场景（通勤/学生党/敏感肌/油皮）
- 正文结构：购买结论 → 为什么（3 点）→ 对比（表格/要点）→ 适合/不适合 → 清单/避坑
- 视觉结构：对比图/细节图/场景图/参数或使用步骤图

### 旅游攻略发布模板

- 标题：城市/路线 + 天数 + 亮点（不踩雷/省钱/亲子友好）
- 正文结构：行程总览 → 每天路线 → 交通与时间 → 预算与住宿 → 必吃必去 → 避坑
- 视觉结构：路线图/地标图/菜单价格/排队时间/机位提示
